# INGEST PLAN

## Goal
Turn the live starter site into a real knowledge hub by introducing the first public-facing asset structure.

## Ingest lanes
1. PDF guides
2. Visual explainers
3. Block diagrams

## Public rule
Every public asset should have:
- title
- short description
- category
- tags
- source/status note
- file location
- website entry

## Order of work
1. Add first library folders
2. Add first starter entries
3. Replace placeholders gradually
4. Polish cards/pages after first real assets are visible
